# 办公小工具整合平台

基于 PySide6 的桌面办公工具整合平台，通过可视化节点编辑器和 AI 辅助，让办公自动化变得简单直观。

## 特性

- 🔧 **节点编辑器** - 可视化工作流设计，拖拽连线即可创建自动化流程
- 📦 **节点包管理** - 从 Git 仓库下载、更新、启用/禁用自定义节点包
- 🤖 **AI 辅助** - AI 助手读取节点列表，辅助设计工作流
- 🔌 **插件系统** - 通过事件系统扩展程序功能
- 💾 **统一存储** - SQLite + SQLAlchemy 本地持久化

## 技术栈

| 层次 | 技术 |
|------|------|
| UI 框架 | PySide6 (Qt) |
| 节点渲染 | QGraphicsView/QGraphicsScene |
| AI 框架 | AgentScope |
| 持久化 | SQLite + SQLAlchemy |
| 插件加载 | importlib 动态导入 |

## 快速开始

### 环境要求

- Python 3.11+
- [uv](https://docs.astral.sh/uv/) 包管理器

### 安装

```bash
# 克隆项目
git clone <repository-url>
cd office

# 安装依赖
uv sync
```

### 启动应用

```bash
uv run office
```

## 基本使用

### 节点编辑器

节点编辑器是本平台的核心功能，你可以：

1. 从左侧节点面板拖拽节点到画布
2. 连接节点端口创建数据流
3. 点击运行按钮执行工作流
4. 保存工作流以便后续使用

### AI 对话

AI 助手可以帮助你：

- 了解可用的节点类型和功能
- 设计和推荐工作流
- 解答使用问题

### 插件管理

在设置面板中可以：

- 查看已安装的插件
- 启用/禁用插件
- 管理插件权限

### 节点包管理

通过节点包管理器可以：

- 从 Git 仓库安装新的节点包
- 更新已安装的节点包
- 启用/禁用/删除节点包

## 项目结构

```
office/
├── src/                    # 源代码
│   ├── core/              # 核心模块（插件、权限、事件）
│   ├── engine/            # 节点引擎
│   ├── ui/                # 用户界面
│   ├── agent/             # AI Agent 集成
│   ├── storage/           # 数据持久化
│   └── nodes/             # 节点包管理
├── plugins/               # 插件目录
├── node_packages/         # 节点包目录
├── workflows/             # 工作流存储
├── config/                # 配置文件
└── data/                  # 数据库文件
```
